## Go Vet

vet 是 golang 中自带的静态分析工具，可以让我们检查出 package 或者源码文件中一些隐含的错误。

go 还有很多质量工具，最常用的就是 ***go imports(来检测引用包的问题)，gofmt(代码格式问题)，golangci-lint(语法检测)***
