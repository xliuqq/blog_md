# Knative

Knative 是以 **Kubernetes** 的一组自定义资源类型（CRD）的方式来安装

- 自动完成代码到容器的构建
- 把应用（或者函数）和特定的事件进行绑定：当事件发生时，自动触发应用（或者函数）
- 网络的路由和流量控制
- 函数的自动伸缩
