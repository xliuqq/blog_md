# Flink API

DataSet 和 DataStream的流程：

1. 获取`execution environment`；
2. 加载/创建数据；
3. 数据转换；
4. 结果存储；
5. 触发程序执行；

data source -> data operations -> data sink



Flink的执行通过 Environment 的 execute 方法，是**根据sink（输出）决定输入以及执行的操作**。

- 两次execute之间，新的sink会被输出；
