    a fast and powerful Message Broker which supports many Cross Language Clients and Protocols and many advanced features.

    Airflow（调度和监控工作流）

    workflow automation and scheduling that can be used to author and manage data pipelines.

    Ambari （Hadoop管理监控工具）

    HDFS、MapReduce、YARN、Hive、Pig、 HBase、Zookeper、Sqoop和Hcatalog。

    Ant（构建工具）

    a Java-based build tool.

    ApiSix（云原生的API服务网关）

    Cloud-native microservices API gateway.

    Archiva（远程资源库maven等管理软件）

    an extensible repository management software that helps taking care of your own personal or enterprise-wide build artifact repository.

    Arrow（跨语言的内存数据格式）

    A cross-language development platform for in-memory analytics.

    AsterixDB（大数据管理系统 (BDMS））

    a scalable, open source Big Data Management System (BDMS).

    Atlas（大数据的元数据管理和数据治理平台）

    Hadoop社区为解决Hadoop生态系统的元数据治理问题而产生的开源项目.

    Avro（行式序列化）

    A data serialization system

    Axis2（Web服务的核心支援引擎）

    a service hosting and consumption framework that makes it easy to use SOAP and Web Services.

    Bahir（Spark/Flink扩展）

    Beam（统一编程模型的批流一体的数据处理）

    Brooklyn（多数据中心的应用程序的蓝图和管理平台）

    a framework for modeling, monitoring, and managing applications through autonomic blueprints.

    Brpc（百度的RPC开源）

    an industrial-grade RPC framework for building reliable and high-performance services.

    Buildr（项目构建系统，类似Maven/Gradle）
